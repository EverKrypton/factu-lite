Arquitectura Cliente-Servidor

🖥️ Cómo funciona

FactuLite funciona con 2 programas separados:

  • FactuLite Server - Se instala en UNA computadora (el servidor)
  • FactuLite Client - Se instala en TODAS las computadoras que van a usar el sistema

🚀 Instalación del Servidor

1

Ejecutar FactuLite-Server-Setup.exe

Instala el servidor en la PC principal. Puede ser cualquier PC de la red.

2

Configurar el Administrador

El asistente te pedirá crear el usuario administrador con su contraseña.

3

Anotar la IP del Servidor

Al finalizar, verás la IP del servidor. ¡Anótala! La necesitarás para conectar los clientes.

Ejemplo: 192.168.1.50

4

Iniciar Sesión

Abre el navegador en: http://localhost:5000
Ingresa con el usuario admin que creaste.

💻 Instalación de Clientes

1

Ejecutar FactuLite-Client-Setup.exe

Instala el cliente en cada PC que usará el sistema (cajas, bodega, ventas, etc.)

2

Conectar al Servidor

Cuando abras el cliente, ingresa la IP del servidor:

IP: 192.168.1.50 | Puerto: 5000
3

Modo Offline

Si el servidor no está disponible, puedes hacer clic en "Trabajar Offline".

Las ventas se guardarán localmente y se sincronizarán cuando el servidor vuelva.

✅ Ventajas del Cliente-Servidor

  • El servidor puede estar en Windows, Linux o Mac
  • Los clientes pueden trabajar offline si el servidor falla
  • Cada usuario ve su historial de ventas
  • El admin ve todo y asigna permisos
  • Hasta 100 usuarios simultáneos
